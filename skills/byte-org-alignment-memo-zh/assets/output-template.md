## Package Modes

- Exec one-pager: 用于高层快速判断和拍板。
- Working memo: 用于负责人和核心团队做深入推进。
- Presentation outline: 用于汇报页面和演讲顺序设计。
- Operating appendix: 用于补充 owner、指标、里程碑、风险和问答。
