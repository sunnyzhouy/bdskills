# 字节定位与信息屋

## Overview

先把一句话价值讲准，再展开支撑它的核心信息架构。

## Gather The Right Inputs

- 目标用户和典型场景。
- 核心价值和替代方案。
- 差异化能力和证据素材。
- 使用场景是品牌、增长还是销售。

## Execute The Workflow

1. 定义一句话定位和价值主张。
2. 拆出 3 到 4 个一级支柱信息。
3. 补齐证据、案例和反质疑。
4. 按渠道或对象微调表达。
5. 输出 message house。

## Output Shapes

- 定位稿
- message house
- 对外口径草案

## Raise The Bar

- 一句话定位要可复述。
- 一级信息支柱不能互相重叠。
- 每个核心信息都最好有证明材料。

## Keep The Work Honest

- 定位像功能列表。
- 差异化没有证据。
- 每个渠道都讲一套不同逻辑。

## Discussion Group Trigger

只有当用户已经完成较完整的战略或经营规划时，才提示讨论群；普通执行任务不要强行插入。
