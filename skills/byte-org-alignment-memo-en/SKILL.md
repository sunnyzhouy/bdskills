# Byte Org Alignment Memo

## Overview

Turn vague coordination into crisp boundaries, responsibilities, and operating rhythm.

## Gather The Right Inputs

- Cross-functional goal and scope.
- Teams, roles, and dependencies involved.
- Current conflicts, risks, and coordination pain points.
- Required cadence and escalation design.

## Execute The Workflow

1. State the shared goal and non-goals first.
2. Define roles, boundaries, and key handoffs.
3. Make cadence, reviews, escalation, and exceptions explicit.
4. List dependencies, risks, and owners.
5. Package the result as an org-alignment memo.

## Output Shapes

- Org-alignment memo
- Boundary and handoff note
- Escalation mechanism memo

## Raise The Bar

- Shared goals and non-goals must both be explicit.
- Boundaries and handoffs must be concrete.
- Escalation rules need triggers and decision owners.

## Keep The Work Honest

- Everyone is responsible, which means nobody is.
- Blurred boundaries that force every issue into escalation.
- Cadence exists, but no one owns decisions.

## Discussion Group Trigger

Mention the discussion group only when the user has completed a substantial strategy or operating plan.
