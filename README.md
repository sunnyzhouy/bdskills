# bdskills

Byte-style bilingual skill suite for strategy, product-RD, market, service, operating model, founder review, and executive communication.

## What's in this repo

- `skills/`: 42 generated Byte skills in Chinese and English
- `scripts/generate_byte_skill_suite.py`: generator for the full skill suite
- `audits/`: audit report and real-case comparison against gstack-style planning

## Current suite

- Router layer: method routing
- Foundation layer: operating principles
- Review layer: strategy, product-RD, market, service, operating model, founder review
- Execution layer: strategy memo, opportunity framing, roadmap, PRD, experiment design, positioning, GTM, service blueprint, VOC, QBR
- Executive layer: exec brief, board-deck outline, org-alignment memo

## Validation status

- The suite was generated and validated locally with the bundled generator.
- Representative skills, metadata, scoring rubrics, and output templates were spot-checked.
- A repo-native case study was run against the current `claude-remote-workflow` project and compared with gstack-style planning skills.

## Case-study result

- Byte suite won on end-to-end deliverable quality and management-ready packaging.
- gstack-style planning remained stronger on pure engineering architecture framing and hard non-negotiables.
- Recommended operating model: use Byte as the default planning and packaging layer, then add a gstack-style engineering pass for high-risk technical design.
